1. Función logística.¶

Puesto que la variable dependiente se trate de una variable categrica, recurrimos a la regresin logstica. Para ello, definimos una funcin logistica o sigmoide, cuyo valor se encontrar siempre entre 0 y 1: $$ \sigma(z) = \frac{1}{1+e^{-z}}$$

Dato que el objetivo es clasificar los datos por probabilidades de manera binaria, en un principio toda probabilidad por debajo de 0.5 computar como "0" y toda aquella que est por encima de 0.5 corresponer a la clase "1".

In [19]:
def funcion_logistica(x):    
    return 1/ (1 + np.exp(-x)) #frmula de la funcin sigmoide

2. Función de coste y gradiente.¶

Esta funcin logstica llevar asociada una funcin de coste, recopilando as el error que se comete al clasificar las observaciones de la muestra: $$J(\theta) = -\frac{1}{m} \sum_{i=1}^{m} [ y^{(i)}log(h_{\theta}(x^{(i)})) + (1 - y^{(i)})log(1 - (h_{\theta}(x^{(i)}))]$$

Se procurar minimzar esta funcin de coste. Dado que nuestra funcin de coste es convexa, nos interesar el valor concreto del modelo que da lugar al mnimo global. Para ello, utilizamos el Descenso del Gradiente, mediante el cual iremos descendiendo por la curva hacia ese mnimo dando "saltos" de manera iterativa.

$$ \frac{\partial J(\theta)}{\partial \theta_j} = \frac{1}{m} \sum_{i=1}^{m} (h_{\theta}(x^{(i)}) - y^{(i)})x_j^{(i)}$$
In [33]:
def costes_computados(theta, x, y):
    m = len(y) #nmero de observaciones
    y_prediccion = funcion_logistica(np.dot(x ,theta)) #utilizamos la funcin logistica
    error = (y * np.log(y_prediccion)) + ((1 - y) * np.log(1 - y_prediccion)) #funcin de coste
    coste = -1 / m * sum(error) #sumatorio del coste
    gradiente = 1 / m * np.dot(x.transpose(), (y_prediccion - y)) #calculamos el gradiente
    return coste[0] , gradiente #valores que queremos que se nos devuelvan

3. Descenso del gradiente.¶

In [39]:
def descenso_gradiente(x, y, theta, alpha, iteraciones):
    costes = [] #guarda los costes
    for i in range(iteraciones):
        coste, gradiente = costes_computados(theta, x, y) #funcin de coste en base a los parmetros
        theta = theta - (alpha * gradiente) #con cada iteracin, usamos la anterior theta y restamos el gradiente por la tasa de aprendizaje
        costes.append(coste) #se va acumulando el coste en costs
    return theta, costes
